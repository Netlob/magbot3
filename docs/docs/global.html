shouldChange(gog, mag)

Checks wether a google appointment should change. This simply dry-runs the updates on a google appointment and checks whether the google appointment has changed then. It is advisable to run this function before patching to google to save api calls for unchanged events. The defaults inside this function are necessary because google doesn't always return default options when getting calendar events. We add them if they are not present so comparing the appointments won't break.
Parameters:
Name Type Description
gog * the google appointment
